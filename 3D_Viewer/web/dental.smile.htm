Introducing Dental.Smile.

Dental.Smile adalah aplikasi pemodelan 3D ortodontik berbasis desktop yang dirancang untuk berjalan di komputer tunggal (stand alone application). Namun, Dental.Smile juga dapat dijadikan sebagai aplikasi jaringan jika database diletakkan di server terpisah dan beberapa komputer terinstall aplikasi Dental.Smile yang terhubung ke database dan file server melalui jaringan. Aplikasi Dental.Smile digunakan oleh seorang dokter gigi maupun beberapa dokter gigi dalam suatu klinik

Made For Dentist Helping Patient.

Sistem Pemodelan 3D Ortodontik diperuntukan bagi dokter gigi atau ortodontis untuk mempermudah pekerjaan mereka dalam melakukan analisa dan diagonasa perawatan gigi pasien. Pemodelan 3D Ortodontik dirancang sebagai sebuah sistem yang dapat memberikan nilai tambah bagi dokter gigi atau ortodontik.

3D Scanner.

Perekaman data digital dari obyek model gigi dilakukan melalui 3D scanner. Dengan teknologi triangulasi menggunakan kamera dan laser, mampu merekam data digital dari obyek gigi. Hasil proses scanning ini masih mentah sehinggan nanti diperlukan pemrosesan segmentasi dan manipulasi lebih lanjut.

3D Segmentation & Manipulation.

Proses segmentasi terbagi menjadi 2 proses, yaitu proses memisahkan gigi dengan gusi, dan proses memisahkan gigi dengan gigi. Setelah didapatkan gigi per gigi, dapat dilakukan proses manipulasi untuk menambahkan brace, wiring, melakukan pengukuran, dll.

Built with Wpf3D library.

Wpf3D and Helix Toolkit Library are powerfull tool.
